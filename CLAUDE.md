# VeraBlock

## Design Context

This project has captured design context for the `impeccable` skill — read these before any UI/UX work:

- `PRODUCT.md` — register (product), platform (web), audience (seniors primary), positioning, brand personality, anti-references, design principles.
- `DESIGN.md` + `.impeccable/design.json` — the visual system (Harbor Navy + Lantern Gold palette, Fraunces/DM Sans/DM Mono type, the "Lit Glass" elevation style), extracted from `frontend/src/styles/tokens.css` and `global.css`.

Re-run `/impeccable document` if the visual system drifts from what's in `DESIGN.md`.
